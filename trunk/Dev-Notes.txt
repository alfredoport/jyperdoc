* Repository Structure

  axserver/
  css/		Contains the style sheet files.
  js/		Contains the JavaScript functions used in JyperDoc.
  axbook/	Contains a xhtml version of the Axiom Book.
  codepress/ 

* OpenAxiom

  Currently OpenAxiom is the platform of choice to run JyperDoc.

  http://open-axiom.org/download.html

* Javascript 2D/3D Plotting Options

   The following libraries are stored in the repository for evaluation:

   * SVG-VML-3D 1.3 http://www.lutanho.net/svgvml3d/index.html

   * Flotr: http://solutoire.com/flotr/

   * Canvas3D: http://dragan.yourtree.org/code/canvas-3d-graph/

   * Photochart: http://code.google.com/p/protochart/
